伪类是对目标元素出现某种特殊的状态应用样式

  • visited:
  • hover: 表示鼠标指针当前停留在该元素上

绝对定位 absolute

绝对定位可以在文档中将元素放在精确的位置

相对定位 relative

类似于绝对定位，但应用相对定位的元素不脱离文档流。

固定定位 fixed

固定定位将元素保持在同一固定位置，即使滚动文档也是如此
